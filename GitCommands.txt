Navigate to Your Project Directory:
cd D:\JavaProgram

Initialize Git Repository:
git init

Add Your Java File to Git:
git add LargestNumberSum.java

If you want to add all the files in the current directory, you can use:
git add .

Commit the Changes:
git commit -m "Add LargestNumberSum program"

To Change branch:
git branch -M main

Add Remote GitHub Repository:
git remote add origin git@github.com:Kalyani-2508/GitTutorial.git

Push the Code to GitHub
git push -u origin main
